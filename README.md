## Getting Started

### Prerequisites

- node

Go to https://nodejs.dev/en/download/ or run

```sh
apk add nodejs npm
```

### Installation

1. Clone the repo

```sh
   git clone https://github.com/AnselOng/urlShortener.git
```

2. Install dependencies

```sh
   npm i
```

3. Run server

```sh
   npm run start
```
